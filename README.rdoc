== Makerbistro

A simple menu application for restaurants

Makerbistro allows you to:

* View a list of all menu items (the menu)

* Create and destroy menu items

* View the ingredients

= To run:

* <tt>bundle install</tt>

* <tt>bundle exec rake db:drop db:create db:migrate db:seed</tt> to populate menu with seeded items

* Enjoy!

<tt>rake doc:app</tt>.
